# AV2
## Manual de instalação
```
cd backend

primeira vez rodando:
npm run start

nas próximas vezes:
npm run dev
```
Abra um novo terminal
```
cd frontend
npm install
npm run dev
```
Primeiro acesso faça login com:
```
usuário: admin
senha: admin123
```
